돌 n개, 1개 or 3개 가져갈 수 있음.
마지막 돌가져가면 이김

### 답
둘이한번에 돌을가지고갈 수 있는 2, 4, 6
이다
이건 베스킨라빈스 게임이랑 비슷하다.
두번째 사람이 의도적으로 4를 맞출 수 있다.
나머지가 1,3이면 먼저한사람이 이기고 나머지가 0,2면 다른사람이 이긴다.
```
import sys

n = int(sys.stdin.readline().strip())
m = n%4
if m == 1 or m ==3:
    print('SK')
else:
    print('CY')
```
